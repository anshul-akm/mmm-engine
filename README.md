# Anshulytics AI – Marketing Mix Modeling & Optimization Suite

🚀 Live App: https://anshuls-mmm-engine.streamlit.app/

## Overview

Anshulytics AI is a prototype end-to-end Marketing Mix Modeling (MMM) and marketing analytics platform built using Streamlit, Python, and machine learning techniques. The application is designed to help marketing and analytics teams evaluate channel effectiveness, optimize promotional spend, simulate business scenarios, and generate actionable insights through an interactive analytics workflow.

The platform automates the complete marketing analytics pipeline — from exploratory data analysis and ETL-driven reporting to ROI analysis, contribution modeling, and budget optimization.

---

# Key Features

## 📊 Marketing Mix Modeling (MMM)

* Adstock transformation for carryover effect modeling
* Hill saturation curves for diminishing returns analysis
* Ridge, Lasso, and ElasticNet regression models
* Marketing channel contribution analysis
* ROI measurement and optimization
* Automated model comparison and selection

---

## 🎯 Scenario Simulation

* Simulate channel-level spend increases or reductions
* Forecast expected sales impact
* Evaluate promotional efficiency under different business scenarios
